# Corporate Majdoor Janta Party (CMJP)

Satirical movement portal for corporate India — anti-nonsense, pro-boundaries, pro-human.

## Stack

- Static HTML / CSS / JavaScript
- [OGL](https://github.com/oframe/ogl) WebGL aurora background

## Local development

Open `index.html` with a local server (ES modules require HTTP):

```bash
npx serve .
```

Or install dependencies for local OGL path (optional):

```bash
npm install
```
